多CONFIG配置

April 1, 2017

点击 Project -> Info -> Configurations 下面的加号, 可以复制一个同样的配置.

点击 Targets -> Build Settings 里的加号, 可以在配置里属性. 添加之后会在最下方看到，然后就可以对不同的配置修改不同的属性值.

获取配置中的属性值

在Info.plist中添加一个别名, 如CFBundleName, 值为配置中的属性名, 如$(BUNDLE_NAME)

tips > 如果出现断点调试变量值都为nil，将Build Settings中Optimization Level设置成None即可。
